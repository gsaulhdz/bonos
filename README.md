# Sistema de Bonos - Mesa de Control

Sistema web para el cálculo de bonos de productividad del personal de mesa de control, rótulos y recibo de proveedores.

## Despliegue en Railway

1. Sube este repositorio a GitHub
2. En Railway, crea un nuevo proyecto desde el repositorio
3. Railway detectará automáticamente la configuración y desplegará la app

## Uso

1. Crear un periodo (quincena o corte)
2. Subir los archivos Excel correspondientes
3. Ir a Reporte de Bonos y seleccionar el periodo
4. Filtrar y consultar resultados

## Archivos soportados

- REPORTE_CHECK_LIST.xlsx
- AFECTACION_DE_BONO.xlsx
- ACTA_DE_INCUMPLIMIENTO.xlsx
- BONO_ROTULOS.xlsx
